## You describe the job. Hermit builds the system.

Say "I need an enterprise sales pipeline." Hermit creates account records, deal records, a Sales Leader agent that runs pipeline reviews, and a live CRM dashboard — all from markdown files in a git repo.

Say "I need a public website." It creates a Website Lead agent, page records, capability descriptions, and renders them into a live site — the one you're reading right now.

The files **are** the application. Not an export. Not a snapshot. The real, working system.
